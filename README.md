## mx-vue-aliplayer

> Aliplayer的Vue组件应用  
> 基于vue-aliplayer作者的更新及优化  
> 暂不支持关于（vid+auth播放)  

## Install

npm install --save mx-vue-aliplayer

## Usage

```js
import MxVueAliplayer from 'mx-vue-aliplayer'

export default {
  components: {
    MxVueAliplayer
  }
}
```
